## About Project
- 클론 코딩 페이지 UI를 기준으로 컴포넌트 분리
- 화면의 크기에 따른 반응형 웹 페이지 구현
- AWS S3, CloudFront를 통한 정적 웹사이트 배포, Github Action을 통한 배포 자동화 처리
<br/>

## 실행 방법
#### 1. repository clone
```
$ git clone https://github.com/MINYUKYUNG/caredit.git
```

#### 2. dependencies install
```
npm install
```

#### 3. project start
```
npm run start
```
<br/>

## 기술 스택
- React 18 + TypeScript
- React Router
- CSS Framwork: styled-components
<br/>

## 디렉토리 구조
| 디렉토리 | 구분 |
| -- | -- |
| assets | 이미지, 폰트 |
| components | 공통 컴포넌트 |
| constants | 전역 사용 상수 |
| pages | 페이지 |
| routes | 리액트 라우팅 |
| styles | 전역 스타일 |
| types | 타입 정의 |
